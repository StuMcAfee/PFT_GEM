# SUIT Cerebellar Atlas

This directory contains atlas files from the [DiedrichsenLab cerebellar_atlases](https://github.com/DiedrichsenLab/cerebellar_atlases) repository.

## Files

| File | Description |
|------|-------------|
| `tpl-SUIT_T1w.nii` | T1-weighted template image in SUIT space |
| `atl-Anatom_space-SUIT_dseg.nii` | Anatomical lobular parcellation (34 regions) |
| `atl-Anatom.tsv` | Label lookup table with region names and colors |

## Regions (34 total)

The anatomical atlas includes cerebellar lobules (I-X) for left, right, and vermis, plus deep cerebellar nuclei:

- Lobules I-IV, V, VI, Crus I, Crus II, VIIb, VIIIa, VIIIb, IX, X
- Vermis subdivisions
- Dentate, Interposed, and Fastigial nuclei (bilateral)

## Source

From the Diedrichsen et al. (2009) probabilistic atlas:
- Repository: https://github.com/DiedrichsenLab/cerebellar_atlases
- Citation: Diedrichsen, J., Balsters, J.H., Flavell, J., Cussans, E., & Ramnani, N. (2009).
  A probabilistic MR atlas of the human cerebellum. NeuroImage, 46(1), 39-46.
